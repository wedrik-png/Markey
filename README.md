# 📌 Markey – Minimalist AHK Bookmark Launcher

Markey is a lightweight bookmark launcher built with AutoHotkey v2. It lets you **save and instantly open** your favorite links using simple hotkeys and a minimal interface.

---

## 🚀 Features

- ✅ Save bookmarks with a hotkey
- ✅ Launch a searchable list of bookmarks
- ✅ Tiny, no-bloat, no external dependencies
- ✅ Comes with a compiled `.exe` — AHK installation needed only for customizations
- ✅ Customizable titles


---

## 🖥️ Hotkeys

| Hotkey | Action |
|--------|--------|
| `Ctrl + Shift + Alt + B` | Open bookmark launcher GUI - to view, add or delete bookamarks |
| `Ctrl + Shift + B`       | Save the active website as new bookmark |

> Hotkeys are customizable in the script if needed.

### 🖼️ Opening Bookmarks
<img width="1919" height="1079" alt="image" src="https://github.com/user-attachments/assets/10ab36a7-8f44-44d7-b148-2b0005db6f4f" />
<p>Press <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Alt</kbd> + <kbd>B</kbd> to open the bookmarks list (hotkey works system-wide). Enter the index number of the bookmark to open it.</p>

### 🖼️ Deleting Bookmarks
<img width="1919" height="1079" alt="image" src="https://github.com/user-attachments/assets/55ba8358-bb22-4718-879e-458088e70498" />
<p>To delete a saved bookmark:

1. Press `Ctrl + Shift + Alt + B` to open the bookmarks GUI.  
2. In the input box that appears, type:

   `<number> delete`

   For example:  
   `3 delete` will delete the 3rd bookmark in the list.</p>
