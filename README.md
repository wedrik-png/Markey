# 📌 Markey – Minimalist AHK Bookmark Launcher

Markey is a lightweight bookmark launcher built with AutoHotkey v2. It lets you **save and instantly open** your favorite links using simple hotkeys and a minimal interface.

---

## 🚀 Features

- ✅ Save bookmarks directly from clipboard
- ✅ Launch a searchable list of bookmarks
- ✅ Tiny, no-bloat, no external dependencies
- ✅ Comes with a compiled `.exe` — AHK installation needed only for customizations
- ✅ Customizable titles


---

## 🖥️ Hotkeys

| Hotkey | Action |
|--------|--------|
| `Ctrl + Shift + Alt + B` | Open bookmark launcher GUI |
| `Ctrl + Shift + B`       | Save the active website as new bookmark |

> Hotkeys are customizable in the script if needed.

---

## 🛠️ Setup

 **Download** the `.exe` from the [Releases](https://github.com/wedrik-png/Markey/releases) section  
   *(Or run the `.ahk` script if you have AutoHotkey v2 installed)*
